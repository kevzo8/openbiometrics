OpenBiometrics ships with **fully open-source models by default**. No commercial restrictions, no API keys required.

## Default Models (Community Tier)

| Model | License | Size | Purpose | Accuracy |
|-------|---------|------|---------|----------|
| YuNet | MIT | 0.3 MB | Face detection | 0.88 AP (WIDER Face) |
| SFace | Apache 2.0 | 37 MB | Face recognition | 99.4% LFW |

## Model Tiers

OpenBiometrics supports three tiers of models. You can mix and match:

### Community (default)
Fully open-source models with permissive licenses (MIT, Apache 2.0). Free for commercial use. No restrictions. This is what you get out of the box.

### Premium
Highest-accuracy models from partner providers. Requires a license key. These models may be proprietary but run on the same OpenBiometrics platform. Partners can register their NIST-ranked algorithms and serve them through the standard API.

### Legacy
InsightFace models (SCRFD, ArcFace, GenderAge) with non-commercial licenses. Kept for backward compatibility. If you need these for production, contact InsightFace for a commercial license.

## Checking Your Models

Every API response includes the `X-OpenBiometrics-Version` header. For detailed model info:

```bash
curl http://localhost:8000/api/v1/admin/health
```

## Choosing Models

Specify models in your configuration:

```python
from openbiometrics import BiometricConfig, FaceConfig

# Auto (default) — picks best available, prefers community
config = BiometricConfig(face=FaceConfig(detector="auto", recognizer="auto"))

# Explicit community models
config = BiometricConfig(face=FaceConfig(detector="yunet", recognizer="sface"))

# Legacy InsightFace models (non-commercial, higher accuracy)
config = BiometricConfig(face=FaceConfig(detector="det_10g", recognizer="w600k_r50"))
```

## Downloading Models

```bash
# Download community models (default)
python download_models.py --module face

# Download all modules
python download_models.py --module all
```

## Registering Custom Models

Partners and enterprises can register their own models:

```python
from openbiometrics.runtime.registry import ModelRegistry, ModelInfo, ModelTier

registry = ModelRegistry(models_dir="./models")
registry.register(ModelInfo(
    name="my_detector",
    filename="my_detector.onnx",
    url="https://models.mycompany.com/detector.onnx",
    description="Custom high-accuracy face detector",
    size_mb=50.0,
    tier=ModelTier.PREMIUM,
    license="Commercial",
    role="detector",
))
```
